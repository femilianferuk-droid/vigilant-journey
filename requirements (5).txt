LEGO 60347 — Мини-кафе (3D инструкция)
=========================================
ЗАВИСИМОСТИ (dependencies)

Нет.
Абсолютно никаких.

Этот проект работает на чистом HTML + CSS + JavaScript (Vanilla JS).
Не требует установки, сервера, npm, CDN, интернета.

ВСЁ УЖЕ ВНУТРИ ФАЙЛА:

- CSS 3D Transforms (встроено в браузер)
- JavaScript (встроено в браузер)
- Никаких Three.js, jQuery, React, Vue, Bootstrap
- Никаких шрифтов из интернета
- Никаких картинок из интернета
- Никаких внешних файлов

КАК ЗАПУСТИТЬ:

1. Скачай файл LEGO_60347_Пошаговая_3D.html
1. Открой двойным кликом (или через «Открыть с помощью» → Браузер)

КОНЕЦ.
